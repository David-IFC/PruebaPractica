# Natalie Portman - Web App Premium (PWA)

Esta es una aplicación web progresiva (PWA) dedicada a la vida y carrera de la actriz Natalie Portman. El proyecto ha sido diseñado con un enfoque en la estética premium, accesibilidad y usabilidad.

## Características Principales

- **Diseño Responsive**: Adaptado para móviles, tablets y escritorio.
- **Semántica HTML5**: Uso correcto de etiquetas para accesibilidad (WCAG) y SEO.
- **PWA Ready**: Incluye `manifest.json` y Service Worker para soporte offline.
- **Estética Minimalista**: Paleta de colores oscuros con acentos dorados para un look sofisticado.

## Tecnologías Utilizadas

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Service Workers para PWA

## Instalación y Uso

1. Clona el repositorio o descarga los archivos.
2. Abre `index.html` en tu navegador.
3. Para la funcionalidad PWA (instalación), sirve los archivos desde un servidor local (ej. Live Server en VS Code).

## Créditos e Imágenes

- **Iconos**: Generados por IA.
- **Contenido**: Biografía y filmografía recopilada de fuentes públicas.
